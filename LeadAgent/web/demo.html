LeadAgent Widget Demo

This page demonstrates the embeddable chat widget. Click the chat bubble in the bottom-right corner to start a conversation.

Embed it on any page with one script tag:

<script
  src="http://localhost:8000/widget/widget.js"
  data-api="http://localhost:8000"
  data-title="Chat with us"
  data-color="#2563eb">
</script>

Configure via data attributes: data-title, data-color, data-position (left/right).
